
# Report Management System

A Full Stack Report Management System that allows users to create, manage, and generate professional reports dynamically with PDF export functionality.

## Tech Stack

**Frontend:** React, TailwindCSS

**Backend:** Node, Express

**Database:** PostgreSQL


## Features
- Create, update, manage reports and integrate Restful APIs
- Dynamic report templates and structured report generation
- Generate downloadable PDF reports using React PDF Renderer
- Scalable backend architecture with PostgreSQL database
- Organized folder structure for maintainability
## Highlights
- Built complete frontend and backend architecture
- Implemented dynamic PDF generation functionality
- Managed database operations using PostgreSQL
